# Game Theory AI: MatPat Research Terminal

Create theories of your favorite games (based on a certain popular youtuber...)

## Tech Stack

**Frontend**
Next.js, React, Tailwind CSS, Framer Motion, Lucide React, Shadcn/UI

**Backend/AI**
OpenAI API (GPT-5.2), Node.js

**Database**
ChromaDB

In Progress: RAG-Enhanced Theory (ChromaDB): Integrating with a vector database (ChromaDB) to retrieve real lore evidence from fandom wikis to inform AI responses for improved accuracy.
